# Banking Frontend

React + Vite single-page app for the Financial Account & Transaction Management System.
See the [root README](../README.md) for full setup instructions, environment variables, and
project overview.

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
npm run lint
```

`VITE_API_BASE_URL` (optional, defaults to `http://localhost:5000/api`) points the app at the
backend API — see `src/config/api.js`.
